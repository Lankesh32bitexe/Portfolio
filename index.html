JARVIS AI

Jarvis is a personal AI assistant I built using LiveKit and Google Realtime API. It can understand and respond in both Hindi and English, perform real-time Google searches, fetch live weather updates, control windows and applications, open files, play YouTube songs, and even use the camera for real-time activity detection.The assistant has modular tools for system control, smart automation, and memory logging to avoid repetitive responses. My aim with this project was to create a voice-based AI system inspired by Jarvis from Iron Man, but fully customized with my own ideas and features..

STUDENT PHOTO

A simple web app where students can enter their roll number to instantly find and view their photo. Built with a clean UI and fast backend lookup, it makes student identification quick and hassle-free..
